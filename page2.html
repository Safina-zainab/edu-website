BASIC CONCEPTS OF OBJECT ORIENTED PROGRAMMING

  1. Class:

    The class contain data and function bundled together under a unit .in other words class is
    a collection of similar objects.when we define a class it just creates templates or skelton.
  2. Object:

    Object are basic unit of opp.They are instances of class,which have data members
    and use various member functions to perform tasks.
  3. Data abstraction:

    Data abstraction refers to providing only essential information to outside world
    and hiding their background deatail.
  4. Encapsulation:

    Encapsulation is the method of combining the data and function inside a class
    This hides the data from beind accessed from the outside world.
    A wrapping of data and function into a single unit is known as encapsulation.
  5. Inheritance:

    Inheritance is the process by which an object inherits properties of objects
    of another class.This increase code reuseability. It allows user to create new
    class from the existing class i.e derived class and base class.
  6. Polymorhism:

    Polymorphism is the ability to exhits in more than onr form. An operation may
    exhibit different behaviour in different instances.The behaviour depends on the type
    of operand provided.
  7. Message Passing:

    A message for an object is a request for execution of a function and
    therefore will involve a function in the receiving object that generates
    the results.
