INTRODUCTION TO OBJECT ORIENTED PROGRAMMING

Object oriented programming can be seen as an extension of
procedural programming The major factor in the invention of object oriented is to remove some of the
flaws encountered in the procedural oriented approach. object oriented programming treats
data as a critical elements in the program development does no allow it to flow freely around the system.
It ties data more closely to the function that operates on it and protects
it from accidental modification from outside world.

FEATURES OF OBJECT ORIENTED PROGRAMMING

  1. More focus is on data rather than functions or procedures
  2. programs are divided into entities known as objects
  3. Data sturctures are desinged such that they characterize objects
  4. functions that operates on data are tied together in data structure
  5. Data is hidden external function cannot access data
  6. Object may communicate with each other through functions
  7. new data items and functions can be easily added whenever essential
  8. follows bottom up approach
  9. increase data security and integrity
  10. view data and function as a single entity
